Remove copilot and copilot-chat plugins on windows:
```bash
Remove-Item $env:LOCALAPPDATA\nvim\lua\plugins\copilot.lua -Recurse -Force
Remove-Item $env:LOCALAPPDATA\nvim\lua\plugins\copilot-chat.lua -Recurse -Force
```
